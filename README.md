# Hangman Game (Python)

A simple Hangman game built in Python using loops, lists, and conditional statements.

## Description
This project is a console-based Hangman game where the player guesses letters to uncover a hidden word.  
It demonstrates core Python concepts such as:
- Lists
- Loops
- Conditional logic
- User input
- String handling

## How the Game Works
1. A random word is selected from a predefined list.
2. The word is displayed as blanks (`_`) for each letter.
3. The player guesses one letter at a time.
4. Correct guesses are revealed in their correct positions.
5. The game continues until the word is fully revealed.

## Technologies Used
- Python 3
- `random` module
